# Objective

Write a Python script that generates and displays a collection of famous Memento Mori
quotations and meditations — reminders of mortality drawn from Stoic philosophy,
classical literature, and historical figures.

# Requirements

- Include at least fifteen distinct Memento Mori quotations or passages
- Each entry should have: the text, the attributed author or source, and a brief
  context note (one sentence)
- The script should randomly select and display one entry each time it is run, or
  optionally display all of them
- Command-line flag: `--all` to print every entry; default is one random selection
- Output should be readable: clear formatting, author attribution visible
- No external dependencies beyond the Python standard library

# Sources to draw from

- Marcus Aurelius, Meditations
- Seneca, Letters and essays
- Epictetus
- Horace (Carpe Diem / Odes)
- Historical practice: Roman triumphs (the slave whispering "memento mori")
- Medieval and Renaissance traditions where relevant

# Output format

Plain Python script in a single file. Include a brief docstring at the top explaining
what it does and how to run it.
